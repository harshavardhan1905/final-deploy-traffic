Goal of This Project!!
Our goal is to provide a real-time traffic-aware route suggestion system that helps users navigate efficiently by considering live traffic conditions. Using FastAPI, TomTom Traffic API, and OpenRouteService, this system dynamically calculates optimal routes, highlights congestion levels with color-coded paths, and displays the final estimated travel time at the destination.
